## Compile

PowerShell/Command Prompt:
```PowerShell
.\mvnw.cmd compile
```

Linux/Unix/Mac
```sh
./mvnw compile
```

## Run

PowerShell/Command Prompt:
```PowerShell
.\mvnw.cmd exec:java -D"exec.mainClass=com.leafriend.ex.whitenoise.App"
```

Linux/Unix/Mac
```sh
./mvnw exec:java -Dexec.mainClass=com.leafriend.ex.whitenoise.App
```
